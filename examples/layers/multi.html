Example: Tile Layers

Aeris.js

The Aeris API provides a set of weather "tile" images, which can be overlaid on a map. Select one or or weather metrics to render the tile layer.

See Aeris.js Tile documentation for information about available tile layers.

Having trouble?
Make sure you've created a apikeys.js file, with a valid Aeris client_id and client_secret.
